# ✖️ Tabuada

Projeto desenvolvido durante meus estudos de JavaScript com o objetivo de praticar estruturas de repetição, manipulação do DOM e geração dinâmica de conteúdo.

## 🚀 Funcionalidades

- Gera a tabuada do número informado
- Exibe os resultados de 1 a 10
- Valida campos vazios

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript

## 📚 Conceitos praticados

- Estruturas de repetição (`for`)
- Manipulação do DOM
- Criação dinâmica de elementos
- Validação de dados
- Interação com o usuário
